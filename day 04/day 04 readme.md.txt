Exercise 1: Rectangle Area Calculator

This program calculates the area of a rectangle based on the length and width entered by the user.

How it works

* The user enters the rectangle’s length.
* The user enters the rectangle’s width.
* The program multiplies the length and width to find the area.
* The calculated area is displayed on the screen.

Concepts Used

* User input with input()
* Data type conversion using float()
* Variables
* Arithmetic operations (*)
* Formatted output using f-strings



Exercise 2: Shopping Cart Program

This program simulates a simple shopping cart where the user can enter an item, its price, and the quantity they want to buy.

How it works

* The user enters the item name.
* The user enters the price of a single item.
* The user enters the quantity.
* The program calculates the total cost by multiplying the price by the quantity.
* Finally, it displays a summary of the purchase and the total amount.

Concepts Used

* User input with input()
* String variables
* Numeric data types (float and int)
* Arithmetic operations
* Formatted output using f-strings
